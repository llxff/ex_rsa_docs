    ExRsa v0.1.0 ExRsa

    Модуль для работы с RSA ключами с 3DES алгоритмом зашифрованных паролей.

    iex> {private_key, public_key} = ExRsa.generate_keys("password")
{"-----BEGIN RSA PRIVATE KEY-----...", <<48, 130, ...>>}
iex> {encrypted_data} = ExRsa.encrypt_public(public_key, "data")
{<<129, 91, ...>>}
iex> {data} = ExRsa.decrypt_private(private_key, "password", encrypted_data)
{"data"}

    Настройки

    По умолчанию для шифрования приватного ключа используется соль, которая генерируется с помощью Erlang модуля crypto и функции strong_rand_bytes. Если вы хотите использовать другой способ генерации, можете определить модуль, который реализует функцию strong_rand_bytes и принимает 1 аргумент (количетсво бит) и указать в настройках config/_env_.exs:

    config :ex_rsa, :random_bytes_generator, MyGenerator

    Summary

    Functions

    decrypt_private(private_key, password, encrypted_data)

    Расшифровывает зашифрованные данные приватным ключом private_key в формате PEM. Результатом является кортеж с расшированными данными

    encrypt_public(public_key, data)

    Шифрует данные data (бинарные или строка) публичным ключом public_key в формате DER. Результатом является кортеж с зашифрованными данными в бинароном формате

    generate_keys(password)

    Генерирует приватный и публичный ключ. Приватный ключ зашифрован 3DES алгоритмом шифрования с паролем password. Результатом выполнения является кортеж с приватным ключом в формате PEM и публичным ключом в формате DER

    Functions

    decrypt_private(private_key, password, encrypted_data)

    Расшифровывает зашифрованные данные приватным ключом private_key в формате PEM. Результатом является кортеж с расшированными данными.

    Examples 

    iex> { private_key, public_key } = ExRsa.generate_keys("password")
iex> { encrypted_data } = ExRsa.encrypt_public(public_key, "data")
iex> ExRsa.decrypt_private(private_key, "password", encrypted_data)
{"data"}
    encrypt_public(public_key, data)

    Шифрует данные data (бинарные или строка) публичным ключом public_key в формате DER. Результатом является кортеж с зашифрованными данными в бинароном формате.

    Examples 

    iex> { _, public_key } = ExRsa.generate_keys("password")
iex> ExRsa.encrypt_public(public_key, "data")
{<<129, 91, ...>>}
    generate_keys(password)

    Генерирует приватный и публичный ключ. Приватный ключ зашифрован 3DES алгоритмом шифрования с паролем password. Результатом выполнения является кортеж с приватным ключом в формате PEM и публичным ключом в формате DER.

    Examples 

    iex> ExRsa.generate_keys("password")
{"-----BEGIN RSA PRIVATE KEY-----...", <<48, 130, ...>>}
