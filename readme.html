    ExRsa

    Модуль для работы с RSA ключами с 3DES алгоритмом зашифрованных паролей.

    Использование

    Генерация ключей

    iex> password = "my password"
iex> {private_key_pem, public_key_der} = ExRsa.generate_keys(password)
{"-----BEGIN RSA PRIVATE KEY-----...", <<48, 130, ...>>}

    Здесь private_key_pem - это приватный ключ в формате PEM, public_key_der - публичный ключ в формате DER.

    Шифрование

    iex> {encrypted_data} = ExRsa.encrypt_public(public_key_der, "data")
{<<129, 91, ...>>}
iex> {data} = ExRsa.decrypt_private(private_key_pem, password, encrypted_data)
{"data"}
